# Tasker

Task tracker CLI app to track your tasks and manage your to-do list.

## Demo

![tasker usage](docs/demo.png "tasker usage")

## Build

```bash
# makefile
make build

# directly
go build -o bin/tasker
```

## Tech Stack

Go, Cobra CLI Framework
